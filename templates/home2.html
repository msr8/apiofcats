Introduction

Hello there! This is a site for cat images and videos. If you want to check out a random picture/video present in the library, you can do so by clicking on the "Random Cat" button which will take you to /random. Once there, you can constantly refresh the page (ctrl+R or cmd+R) to load new cat pics/vids :). There are currently {{shit_to_be_filled_out_in_python}} images and videos present in the evergrowing library. Endpoints

NOTE: BODY arguments are discarded/ignored if ANY URL arguments are given, even invalid ones

    /random
  • Info

    Description: Shows a random cat image/video
    Methods: GET and POST
    Parameters:

    Location Name Type Default Description Example
    URL & BODY ext string jpg,jpeg,png,mp4 The extension of the image to be returned. You can allow multiple extensions by seperating them with a comma /random?ext=png,jpg,jpeg
    {'ext' : 'png,jpg,jpeg'}
    URL & BODY max_size int None (no limit) The maximum size of image to be returned (in bytes) /random?max_size=8000000
    {'max_size' : 8000000}
    URL & BODY redirect bool False Wether to redirect the user to the /library endpoint /random?redirect=True
    {'redirect' : True}


    Sample requests:

    # To get a random image/video
    curl {{DOMAIN}}/random
    # To get a random video
    curl {{DOMAIN}}/random?ext=mp4
    curl {{DOMAIN}}/random -d ext=mp4
    # To get a random image/video with a maximum size of 8MB
    curl {{DOMAIN}}/random?max_size=8000000
    curl {{DOMAIN}}/random -d max_size=8000000

    /api/random
  • Info

    Description: Basically the API version of the /random endpoint
    Methods: GET and POST
    Parameters:

    Location Name Type Default Description Example
    URL & BODY ext string jpg,jpeg,png,mp4 The extension of the image to be returned. You can allow multiple extensions by seperating them with a comma /api/random?ext=png,jpg,jpeg
    {'ext' : 'png,jpg,jpeg'}
    URL & BODY max_size int None (no limit) The maximum size of image to be returned (in bytes) /api/random?max_size=8000000
    {'amount' : 5}
    URL & BODY amount int 1 amount of media returned /api/random?amount=5
    {'redirect' : True}


    Sample requests:

    # To get a random image/video
    curl {{DOMAIN}}/api/random
    # To get a random video
    curl {{DOMAIN}}/api/random?ext=mp4
    curl {{DOMAIN}}/api/random -d ext=mp4
    # To get a random image/video with a maximum size of 8MB
    curl {{DOMAIN}}/api/random?max_size=8000000
    curl {{DOMAIN}}/api/random -d max_size=8000000
    # To get 5 random images/videos
    curl {{DOMAIN}}/api/random?amount=5
    curl {{DOMAIN}}/api/random -d amount=5
