Introduction

Hello there! This is a site for, well, cat images and videos. There are currently {{shit_to_be_filled_out_in_python}} images and videos present in the library. This website is open source (https://github.com/msr8/catapi) so feel absolutely free to check out the code yourself, report an issue, or contribute towards it :). I got all my images/videos through public sources (mostly reddit) and I DO NOT claim ownership of any of them. If you want to check out the random cats I have stored, you can do so by clicking on the "Random Cat" button which will take you to ({{DOMAIN}}/random). Sometimes what I like to do is just visit /random and just keep refreshing and enjoying all the cats


Endpoints

/random
Info

Methods: GET and POST

Parameters:
Location Name Type Default Description Example
URL & BODY ext string jpg,jpeg,png,mp4 The extension of the image to be returned. You can allow multiple extensions by seperating them with a comma /random?ext=png,jpg,jpeg
{'ext' : 'png,jpg,jpeg'}
URL & BODY max_size int None (no limit) The maximum size of image to be returned (in bytes) /random?max_size=8000000
{'max_size' : 8000000}
URL & BODY redirect bool False Wether to redirect the user to the /library endpoint /random?redirect=True
{'redirect' : True}


Sample requests:

# To get a random image/video
curl {{DOMAIN}}/random
# To get a random video
curl {{DOMAIN}}/random?ext=mp4
curl {{DOMAIN}}/random -d ext=mp4
# To get a random image/video with a maximum size of 8MB
curl {{DOMAIN}}/random?max_size=8000000
curl {{DOMAIN}}/random -d max_size=8000000
