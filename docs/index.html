Endpoints

/random
Info

Methods: GET

Parameters:
Location Name Type Default Description Example
URL ext string jpg,jpeg,png,mp4 The extension of the image to be returned. You can allow multiple extensions by seperating them with a comma /random?ext=png,jpg,jpeg
URL max_size int None (no limit) The maximum size of image to be returned (in bytes) /random?max_size=8000000
URL redirect bool False Wether to redirect the user to the /library endpoint /random?redirect=True


Sample request:

curl {{DOMAIN}}/random
